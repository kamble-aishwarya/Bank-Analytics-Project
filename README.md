# Bank Analytics Project

This repository contains my complete **end-to-end Bank Data Analytics Project**.  
It demonstrates my skills in **SQL, Excel, Power BI, and Tableau**, covering the full data analysis workflow from data cleaning to insights and dashboard development.

---

## Project Structure

### **1. Data Understanding & Excel Work**
Includes:
- Raw banking dataset  
- Data cleaning steps (removing duplicates, handling nulls, formatting corrections)  
- Exploratory Data Analysis (EDA) in Excel  
- Pivot tables, charts, and summary insights  
- Initial KPIs and customer/loan trend observations  

---

### **2. SQL Analysis**
Contains:
- Complete SQL query scripts  
- Joins, aggregations, subqueries, window functions  
- Customer segmentation & behavior analysis  
- Loan performance and repayment patterns  
- Transaction trend analysis and anomaly detection  

---

### **3. Power BI Dashboard**
Includes:
- PBIX dashboard file  
- Data model and relationships  
- DAX measures for KPIs  
- Visual insights on:
  - Customer segmentation  
  - Loan performance  
  - Revenue contribution  
  - Monthly/quarterly transaction patterns  
- Interactive slicers for dynamic insights  

---

### **4. Tableau Visualization**
Includes:
- Tableau packaged workbook (TWBX)  
- Dashboard showing customer segmentation, loan insights & trends  
- Filters and interactive views  
- LOD expressions and calculated fields  
- Final dashboard screenshots (if included)  

---

## Tools Used
- **Excel** – Data Cleaning, EDA, Pivot Tables, Charts  
- **SQL (MySQL)** – Joins, Window Functions, Aggregations  
- **Power BI** – DAX, Data Modeling, Interactive Dashboards  
- **Tableau** – Visual Analytics, LOD Expressions, Storytelling Dashboards  

---

## Project Goal
To perform a full-cycle analysis of banking data:
- Understand business objectives  
- Clean and prepare raw datasets  
- Use SQL to extract analytical insights  
- Build BI dashboards (Power BI & Tableau)  
- Deliver final insights & business recommendations
